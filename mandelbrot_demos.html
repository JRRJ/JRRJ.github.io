Mandelbrot Set Experiments

The Mandelbrot Set is the set of complex numbers c for which the polynomial zn + 1 = zn2 + c remains bounded (as n goes from zero to infinity). This means the number 1 is not part of this set, for example.

    For c = 1:
  • n = 0, 02 + 1 = 1
  • n = 1, 12 + 1 = 2
  • n = 2, 22 + 1 = 5
  • n = 3, 52 + 1 = 26
  • ...

Clearly, for c = 1, the polynomial is not bounded, instead it increases infinitely. Complex numbers close to the edge of the Mandelbrot set may seem bounded, with many iterations required to tell if they are or not.

This video by Numberphile has a great easy to understand explanation.

Demonstrations

The method used below to plot the Mandelbrot Set is an implementation of the escape time algorithm. Each pixel is given a point coordinate. The number of iterations it takes for a point to exceed a magnitude of 2 determines the color of the point on the image. If the maximum number of iterations is reached and the magnitude is less than 2, the point is considered part of the set and colored accordingly. Fewer iterations makes for a blocky, inaccurate graph with fewer colors, but the tradeoff is that very many iterations will be very slow.

Increasing Iterations

Each click increases the number of maximum iterations used.

Zoom(30 Iterations)

This demonstration is locked at a maximum of 30 iterations, giving a good tradeoff of accuracy and speed. Clicking a point on the image will zoom in by 2x, centered on the point clicked.

Zoom(100 Iterations)

Same as the previous demo, but with 100 iterations for a much more accurate image. However, processing time is higher.
